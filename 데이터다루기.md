# machinelearningwithpython
파이썬을이용한통계적머신러닝/자유아카데미

#데이터다루기 
- 목표:
    - 머신러닝 알고리즘에 주입할 데이터 준비 방법
    - 데이터 형태가 알고리즘에 미치는 영향
- 훈련 세트와 테스트 세트:
    - 지도학습과 비지도학습
    - 훈련데이터 : 입력(데이터) + 타깃(정답)
- 노트:
    - 배열의 인덱스 :배열의 위치
    - 파이썬 리스트는 인덱스 외에도 슬라이싱이라는 특별한 연산자 제공
    - 슬라이싱 연산을 사용하면 인덱스의 범위를 지정하여 원소를 여래 개 선택할 수 있다.
    - 인덱스 0부터 시작
